# DuoTrack

DuoTrack is a couple-focused companion app designed to help two people stay connected, organized, and in sync through shared activities, tracking features, and real-time interactions.

Built with a modern fullstack architecture using Next.js and Supabase, DuoTrack combines authentication, realtime systems, database functions, and interactive UI into a simple but meaningful experience for couples.

## Features

* Couple-based account system
* Realtime synchronization
* Shared tracking & activity features
* Secure authentication
* Modern responsive interface
* Supabase backend integration
* Edge Functions & PostgreSQL Functions support

## Tech Stack

* Next.js
* React
* Supabase
* PostgreSQL
* TypeScript
